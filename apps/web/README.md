# Web App

This directory contains the Next.js dashboard and API for the DIB Personal Finance Tracker.

Use the root documentation for setup, architecture, auth, and deployment details:

- [`../../README.md`](../../README.md)

Key responsibilities in this app:

- Supabase Auth login / reset flow
- `app_users` authorization checks
- DIB ingest webhook
- dashboard analytics
- transaction, category, and rule management
